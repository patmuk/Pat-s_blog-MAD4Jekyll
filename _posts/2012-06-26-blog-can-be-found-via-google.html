Vor 18 Tagen habe ich diesen Blog ins Leben gerufen. Heute wurde ich durch Google informiert, dass der Blog gecrawlt wurde, also durch Google auffindbar ist. Ohne jegliche Bitte um Einbindung.
Woher ich das weis? Ich monitore die Verwendung meines Namens per http://www.google.com/alerts.
Kann ich nur empfehlen - so bemerkt man, wenn etwas über einen veröffentlicht wird.18 days ago I started this blog. Today Google notified me that my Blog has been crawled and can be found via a google search. Without any action on my side to be registered.
Well, they did not exactly notified my, but I got informed by monitoring the usage of my name in the web.
This can be done using http://www.google.com/alerts, which I can highly recommend!
